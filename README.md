# Bambu Monitor 在线烧录

这是 Bambu Monitor for ESP32-S3 DualEye 的在线烧录页面仓库。

访问 GitHub Pages 后，使用桌面版 Chrome 或 Edge，通过 USB 数据线连接设备并点击页面中的“安装所选固件”即可刷写。
